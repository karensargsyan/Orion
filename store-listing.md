## Title
Orion - AI Browser Assistant

## Short Description (132 chars max)
AI browser assistant that runs locally or via cloud APIs. Reads pages, fills forms, researches the web. Private. Fast. Autonomous.

## Detailed Description

Orion is a fully autonomous AI browser assistant. It can read any page, fill forms, research the web, draft replies, and remember context across sessions.

Run it 100% locally with LM Studio (zero data leaves your machine) or connect your own API key for Gemini, OpenAI, or Anthropic.

WHAT ORION CAN DO

- Read and understand any webpage
- Fill out forms intelligently (visa applications, registrations, checkouts)
- Research the web: opens tabs, reads results, synthesizes findings
- Draft and review emails
- Remember context across sessions and domains
- Store credentials securely in an encrypted vault
- Detect phishing and unsafe pages
- Execute multi-step browser tasks (up to 25 actions per request)

HOW IT WORKS

1. Click the Orion icon on any tab
2. A sidebar opens with a chat interface
3. Type your request and Orion takes action
4. It reads the page, decides what to do, and executes autonomously

Each tab gets its own color-coded group. Orion only accesses tabs you explicitly open it on.

AI PROVIDERS

- Local (LM Studio): Free, private, runs on your machine
- Google Gemini: Fast cloud AI with free tier
- OpenAI (GPT-4o): Industry-standard cloud AI
- Anthropic (Claude): Advanced reasoning

You provide your own API key. No subscription. No middleman.

PRIVACY FIRST

- Local mode: absolutely nothing leaves your computer
- Cloud mode: data goes directly to your chosen provider only
- Zero telemetry, zero analytics, zero tracking
- Tab isolation: only reads tabs in its own group
- Vault uses AES-GCM encryption with PIN protection
- Open source: audit the code yourself

PERMISSIONS EXPLAINED

- Debugger: Enables reliable browser interactions via Chrome DevTools Protocol
- Storage: For settings, memory, and encrypted vault
- Tabs/SidePanel: For the sidebar chat interface
- All URLs: To read any page you open in Orion (restricted to its tab group only)

REQUIREMENTS

- Chrome 114+ (or Chromium-based browser)
- For local AI: LM Studio (free, open source) with any compatible model
- For cloud AI: Your own API key from Gemini, OpenAI, or Anthropic
